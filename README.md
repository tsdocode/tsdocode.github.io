# tsdocode Portfolio

This is a modern, responsive portfolio website built with React, showcasing my professional experience, projects, and skills as a Machine Learning Engineer.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Clean, professional interface with smooth animations
- **Work Experience**: Timeline of professional experience with detailed descriptions
- **Open Source Contributions**: Showcase of contributions to open source projects
- **Social Media Integration**: Links to GitHub, LinkedIn, Twitter, and email contact

## Tech Stack

- **React**: Frontend library for building the user interface
- **React Router**: For navigation between different sections
- **Framer Motion**: For smooth animations and transitions
- **Tailwind CSS**: For styling and responsive design

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/tsdocode/portfolio.git
   cd portfolio
   ```

2. Install dependencies
   ```bash
   npm install
   # or
   yarn install
   ```

3. Start the development server
   ```bash
   npm start
   # or
   yarn start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser
